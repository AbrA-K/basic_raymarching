Description

This is a small raymarching sandbox for you to play around in. It has a couple of demos to show some basic features. If you want to explore more, you can change to the full ui, which will crowd the screen a lot (hence the default minimal ui). To give you some interesting knobs to turn, I recommend playing around with:

in Global Settings:
- Termination distance
- glow_color (notice: it's default 100% transparent)
- glow_range
- smooth intersection
- intersection method

in Object Settings
- rotation over time
- translation over time

Game

WARNING:
The game binary can be quite large and running it might take a bit of processing power.
I recommend to load the game on a non-metered connection while on a laptop/desktop pc.
